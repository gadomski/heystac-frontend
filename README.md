## Developing

Get [yarn](https://yarnpkg.com/) and [uv](https://docs.astral.sh/uv/getting-started/installation/).
Then:

```shell
scripts/setup
```

To start the development server:

```shell
scripts/start
```

To run all tests:

```shell
scripts/test
```

To run all linters and format checkers:

```shell
scripts/lint
```
